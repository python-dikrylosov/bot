## 📖 Описание
**Dikrylosov Quantum Core** — это уникальная исследовательская платформа, объединяющая два направления:
1.  **Квантовая симуляция (Q25DC):** Высокоточный симулятор на 25 кубитах, разработанный в 2016 году для моделирования процессов принятия решений через квантовую интерференцию.
2.  **Локальный AI-ассистент (Dikrylosov AI):** Автономная языковая модель (на базе Qwen3-Coder), оптимизированная для работы на потребительском железе (16GB VRAM + 128GB RAM).

Уникальность системы заключается в режиме **«Самоанализ»**: в свободное от задач время ИИ генерирует вопросы из «Толкового словаря живого великорусского языка» В.И. Даля, прогоняет их через квантовый генератор энтропии и дообучается на собственных ответах, формируя уникальную базу знаний без участия человека.

---

## ✨ Ключевые возможности

### 🔮 Квантовое ядро (Quantum Core)
*   **25 Кубитов:** Полная симуляция вектора состояния ($2^{25}$ амплитуд) на классическом CPU/GPU.
*   **Принятие решений:** Использование квантовой суперпозиции для выбора оптимального пути в задачах оптимизации.
*   **Генерация энтропии:** Истинно случайные числа для криптографии и стохастического обучения нейросетей.
*   **Наследие 2016:** Архитектура, заложенная 10 лет назад, доказавшая свою экономическую эффективность (рост производительности в 4x к 2026 году).

### 🧠 AI-Ассистент с Самообучением
*   **Локальность:** Полная приватность данных. Работает offline на вашем ПК.
*   **Режим "Самоанализ" (Idle Mode):**
    *   Когда пользователь неактивен, система автоматически берет слова из Словаря Даля (от А до Я).
    *   Формирует философские или технические вопросы на основе этих понятий.
    *   Использует квантовые параметры для вариативности вопросов.
    *   Генерирует ответы и сохраняет их в базу знаний (`logs/self_learning.json`).
*   **API Совместимость:** Поддержка формата Anthropic API (`/v1/messages`) для легкой интеграции с внешними инструментами (IDE, скрипты).
